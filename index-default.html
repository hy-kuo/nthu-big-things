您的網站。

使用 iWeb、Pages 和許多其他 Mac OS X 應用程式來輕易快速地建立並發佈網站。

從 Mac 建立並發佈個人網站再簡單也不過了。要發佈網站時，亦可輕鬆完成。

只要打開“系統偏好設定”，按“共享”一下，然後勾選“網頁共享”。

大功告成！現在可以立即在家用或公司的專用網路連接您的網站。

若連接 Internet，各地的朋友都可以連上您的網站。只要將“共享”偏好設定中顯示的位址寄給他們即可。

Apache Power

“網頁共享”內建於 Apache 網頁伺服器（Mac OS X 隨附的業界標準技術）。如需更多 Apache 網頁伺服器的相關資訊，請參閱 Apache 手冊。
